Agents

Introduction to Agents

Agents use an LLM to determine which actions to take and in what order. An action can either be using a tool and observing its output, or returning a response to the user.

In this lesson, you'll learn how to use LangChain to create agents that can dynamically select and use tools based on user input.

Setting Up

First, let's install the necessary packages and import the modules we'll need.

pip install langchain openai
import os
from langchain.agents import AgentType, initialize_agent, load_tools
from langchain.llms import OpenAI

# Set your OpenAI API key
os.environ["OPENAI_API_KEY"] = "your-api-key"

Creating a Simple Agent

Let's create a simple agent that can use search and calculator tools to answer questions. 

llm = OpenAI(temperature=0)

# Load some tools to use
tools = load_tools(["serpapi", "llm-math"], llm=llm)

# Create an agent that uses these tools
agent = initialize_agent(
    tools, 
    llm, 
    agent=AgentType.ZERO_SHOT_REACT_DESCRIPTION,
    verbose=True
)

Now we can run the agent with a query: 

agent.run("What was the high temperature in SF yesterday in Fahrenheit? What is that number raised to the .023 power?")

Agent Types

LangChain provides several agent types for different use cases:

  • ZERO_SHOT_REACT_DESCRIPTION: Uses the ReAct framework to determine which tool to use based solely on the tool's description.
  • STRUCTURED_CHAT_ZERO_SHOT_REACT_DESCRIPTION: A more structured version that works well with chat models.
  • OPENAI_FUNCTIONS: Uses OpenAI's function calling capability for more reliable tool usage.
  • SELF_ASK_WITH_SEARCH: Designed specifically for search-based queries.

Custom Tools

You can create custom tools for your agents to use: 

from langchain.agents import Tool
from langchain.utilities import GoogleSearchAPIWrapper

search = GoogleSearchAPIWrapper()
tools = [
    Tool(
        name="Google Search",
        description="Search Google for recent information",
        func=search.run
    )
]

agent = initialize_agent(
    tools, 
    llm, 
    agent=AgentType.ZERO_SHOT_REACT_DESCRIPTION,
    verbose=True
)

Agent with Memory

You can add memory to agents to maintain context across multiple interactions: 

from langchain.memory import ConversationBufferMemory

memory = ConversationBufferMemory(memory_key="chat_history")
agent = initialize_agent(
    tools, 
    llm, 
    agent=AgentType.CONVERSATIONAL_REACT_DESCRIPTION,
    memory=memory,
    verbose=True
)

Key Takeaways

  • Agents use LLMs to determine which actions to take based on user input
  • Tools provide specific capabilities to agents
  • Different agent types are optimized for different use cases
  • You can create custom tools to extend agent capabilities
  • Adding memory allows agents to maintain context across conversations
